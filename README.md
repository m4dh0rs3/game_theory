# Game theory: XY

Each round, all `n` actors choose `X` or `Y`. Rules on the set of the choice of all actors descide, how much each actor earns or looses.

### Rules for `n = 4`

Set    | `X`   | `Y`
------ | ----- | ----
`XXXX` | `-10` |   –
`XXXY` | `+10` | `-30`
`XXYY` | `+20` | `-20`
`XYYY` | `+30` | `-10`
`YYYY` |   –   | `+10`
